Operasi Matriks

Pelajari berbagai operasi yang dapat dilakukan pada matriks

Penjumlahan

A + B = C

Dua matriks dapat dijumlahkan jika memiliki ukuran yang sama

[a b] + [e f] = [a+e b+f]
[c d] + [g h] = [c+g d+h]

Pengurangan

A - B = C

Pengurangan matriks dilakukan elemen per elemen

[a b] - [e f] = [a-e b-f]
[c d] - [g h] = [c-g d-h]

Perkalian

A × B = C

Perkalian matriks memerlukan kesesuaian ukuran

m×n × n×p = m×p

Transpose

Aᵗ

Menukar baris dan kolom matriks

[a b]ᵗ = [a c]
[c d] = [b d]

Determinan

det(A)

Nilai skalar yang memberikan informasi tentang matriks

det([a b]) = ad - bc
([c d])

Invers

A⁻¹

Matriks yang membalikkan efek dari matriks asal

A × A⁻¹ = I
